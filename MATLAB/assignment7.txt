        character vectors | string scalars
      <a href="matlab:web /MATLAB/help/matlab/ref/who.html#bqup003-20-filename">filename</a> - Name of MAT-file
        character vector | string scalar

    Output Arguments
      <a href="matlab:web /MATLAB/help/matlab/ref/who.html#bqup003-20-C">C</a> - List of variables
        cell array of character vector

    Examples
